About Me


Hello there! My name is Kevin Lutz, welcome to my portfolio site. I am mainly a web developer but I create software applications as well. I am mostly self-taught, having started teaching myself how to code at around age 13. I am currently studying computer science at Pace University in Westchester, New York. I am proficient in Java, JavaScript (including jQuery), CSS, and HTML. Along with attending college, I have also earned my Responsive Web Design Certification and JavaScript Algorithms and Data Structures Certification through freeCodeCamp, which is an online coding course. My ultimate goal is to one day work for Google as a software engineer.
